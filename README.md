# Dungeon Survivor

**Dungeon Survivor** é um jogo 2D top-down desenvolvido em **Godot 4**, inspirado em jogos de sobrevivência por ondas, com movimentação rápida, escolha de classes, inimigos variados e sistema de evolução durante a partida.

## Sobre o jogo

O jogador entra em uma dungeon infestada por monstros e precisa sobreviver ao máximo possível enfrentando ondas de inimigos. Durante a partida, é possível ganhar experiência, subir de nível e escolher melhorias para fortalecer o personagem.

O jogo possui duas classes jogáveis:

* **Duelista**: personagem focado em combate corpo a corpo.
* **Maga**: personagem focada em ataques à distância.

## Funcionalidades

* Sistema de movimentação top-down
* Dash e cambalhota com uso de stamina
* Seleção de classes
* Sistema de ondas de inimigos
* Inimigos com comportamentos diferentes
* Boss
* Sistema de experiência e level up
* Escolha de upgrades durante a partida
* HUD com vida, stamina, experiência, level, wave e contador de inimigos
* Menu inicial
* Tela de game over
* Suporte a tela cheia

## Tecnologias utilizadas

* **Godot Engine 4**
* **GDScript**
* Pixel art 2D
* Git e GitHub para versionamento

## Download

A versão executável para Windows está disponível na aba **Releases** deste repositório.

Baixe o arquivo `.zip`, extraia a pasta e execute `DungeonSurvivor.exe`.

## Como executar o projeto

1. Baixe ou clone este repositório:

```bash
git clone https://github.com/WeebBryan/dungeon-survivor.git
```

2. Abra o projeto no **Godot 4**.

3. Execute a cena principal do jogo.

## Controles

| Ação       | Tecla/Controle          |
| ---------- | ----------------------- |
| Movimento  | W, A, S, D              |
| Mirar      | Mouse                   |
| Atacar     | Botão esquerdo do mouse |
| Dash       | Shift                   |
| Cambalhota | Espaço                  |
| Tela cheia | F11                     |
